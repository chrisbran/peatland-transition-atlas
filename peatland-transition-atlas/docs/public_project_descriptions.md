# Public Project Descriptions

## One-line description

Interactive literature-driven atlas on peatland rewetting, GHG mitigation and transition-compatible agricultural land use.

## Short portfolio description

The Peatland Transition Atlas translates scientific literature on agricultural peatland rewetting into an interactive evidence atlas. It compares international evidence regions, visualises transition pathways between drained farming and restoration, and explores which wetter land-use options may be plausible for South German dairy, forage and biogas landscapes.

## More detailed portfolio description

Drained agricultural peatlands are climate-relevant production landscapes. Rewetting can reduce emissions, but the practical transition is not binary: between drainage-based farming and full restoration lies a spectrum of wetter land-use pathways. This prototype turns curated literature into an exploratory web atlas that connects hydrology, GHG mitigation, farm compatibility, market maturity, adoption barriers and regional transfer hypotheses.

The project is also designed as an AI-agent learning workflow. Research, data, visualisation, editorial, QA and release tasks are separated into agent roles with explicit inputs, outputs and acceptance criteria.

## Caution text

This is a literature-driven prototype, not a validated regional decision-support model. Scores are qualitative expert codings; evidence-map points are approximate visual anchors; and South Germany fit is a transfer hypothesis requiring local hydrological-economic validation.
