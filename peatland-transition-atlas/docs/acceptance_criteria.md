# Acceptance Criteria

## Scientific acceptance

- Claims are no stronger than the underlying evidence.
- Literature reviews are not treated as site-specific empirical evidence.
- South Germany transfer is framed as a hypothesis.
- Hydrology, GHG mitigation, farm economics and adoption are connected.
- Caveats are visible.

## Data acceptance

- Every dataset has a source or origin note.
- Public data sources have licence notes.
- Qualitative scores are labelled as expert coding.
- Approximate coordinates are labelled as visual anchors.
- No confidential or copyrighted source material is committed.

## Visualisation acceptance

- Visual encodings match data meaning.
- Qualitative scores do not look like measured precision.
- Uncertainty is visible.
- The site works on desktop and acceptable mobile width.
- No major browser console errors.

## Repository acceptance

- README explains the project.
- Folder structure is clear.
- GitHub Pages can serve the prototype.
- `.gitignore` excludes raw/licence-sensitive data.
- Docs explain methodology and limitations.
