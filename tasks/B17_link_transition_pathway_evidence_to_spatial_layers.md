# Task B17 — Link Transition Pathway Evidence to Spatial Layers

## Goal

Connect the literature-derived transition pathways more explicitly to the spatial story.

## Rationale

The atlas now has a clear spatial chain:

1. global peatland extent,
2. country emissions pressure,
3. Europe context,
4. Germany implementation layer,
5. Baden-Württemberg planning boundary.

The next step is to make the transition-pathway evidence feel connected to this chain instead of appearing as a separate literature module.

## Work items

1. Add a compact pathway matrix: wet grazing, paludiculture biomass, water-level management, governance/adoption.
2. For each pathway, add:
   - evidence strength,
   - spatial fit,
   - implementation constraints,
   - data gap.
3. Add a visual link from South Germany/BW section to the pathway cards.
4. Add caveat: pathway fit is evidence-informed, not a site-level recommendation.

## Acceptance criteria

- Evidence cards are connected to the map storyline.
- The user understands which pathways are relevant where and why.
- The atlas remains a prototype, not a prescriptive planning tool.
