# B160 - Proposed Main-Flow Outline

Date: 2026-07-01

## Premium-Hauptfluss

### 1. Hook

**Moorbodenschutz braucht räumliche Orientierung.**

Kurzversprechen:

> Wiedervernässung ist nicht nur eine ökologische Maßnahme. Sie verändert Nutzung, Betriebe, Wasser und Wertschöpfung.

Scope als kurzer, einklappbarer Hinweis; nicht als langer Warnblock.

### 2. Warum Karten nötig sind

Ein starker Sticky-Zoom statt vieler erklärender Kartenabschnitte:

1. Moore sind global kleinflächig, aber relevant.
2. In Europa/Deutschland wird aus Klima eine Planungskulisse.
3. In Baden-Württemberg wird die Frage regional.
4. In Oberschwaben trifft sie auf konkrete Nutzung.

### 3. Oberschwaben als Beleg

- statische Karte: Layerlogik
- Felt: interaktive Details
- Bilanz: `~19.900 ha`, `~82 % Grünland`, `~16 % Ackerland`

### 4. Warum die Schnittmenge keine Lösung ist

Ein kompakter Transformations-/Governance-Moment:

- Fläche zeigt Prüfbedarf
- Wasser verbindet Parzellen
- Betriebe und Zuständigkeiten entscheiden mit

### 5. Climax: Die Kette hinter dem Feld

Eine neue Editorial-Grafik ersetzt oder dominiert Scorecard/Matrix:

> Bis zur Ernte ist vieles anschlussfähig. Danach wird es eng.

Visuelle Logik:

```text
Feld / Wasser / Ernte  →  Logistik  →  Verarbeitung  →  Standards  →  Abnahme / Markt
stabiler               →  im Aufbau →  brüchig       →  brüchig    →  eigentliche Engstelle
```

### 6. Kicker

**Der Hebel verschiebt sich von der Fläche zur Kette.**

Kurz, kein weiterer Methodenblock.

Alles, was fachlich nötig ist, aber den Hauptfluss bremst:

- Datenbasis
- Rechte
- Methode
- Nicht-Eignungskarten-Grenzen
